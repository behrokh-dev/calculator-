# 💜 Calculator

A modern, responsive calculator built with **HTML**, **CSS**, **JavaScript**, and **Tailwind CSS**.

This project was created as a JavaScript practice project to improve DOM manipulation, event handling, and application logic. Instead of using `eval()`, the calculator performs calculations using custom logic and a **flag-based state management** approach.

## 💡 How It Works

The calculator keeps track of:

- First number
- Operator
- Second number

A **flag** is used to determine whether the user is entering the first or second operand. Once the equals button is pressed, the appropriate operation is performed without relying on JavaScript's `eval()` function.

This approach makes the logic easier to understand and safer for beginners learning JavaScript.
